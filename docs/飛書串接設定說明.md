# 飛書展會雷達串接

## 飛書端設定

1. 在飛書開放平台建立企業自建應用。
2. 開啟電子試算表讀取與寫入、雲文件讀取權限。
3. 發布並安裝應用到企業。
4. 將目標飛書試算表分享給該應用，允許編輯。
5. 若需要群組提醒，在群組新增自訂機器人並取得 Webhook。

## 本機設定

1. 複製 `feishu-sync-config.example.json`，命名為 `feishu-sync-config.json`。
2. 填入：
   - `app_id`
   - `spreadsheet_url`：開啟「展會追蹤」工作表時的完整網址
   - 選填 `webhook_url`
3. 不要將 `app_secret` 寫入公開檔案。建議設定環境變數 `FEISHU_APP_SECRET`。

同步工具會從完整網址自動辨識 `spreadsheet_token` 與 `sheet_id`。

## 執行

```bash
export FEISHU_APP_ID="cli_aaa080ae0f78de18"
export FEISHU_APP_SECRET="cMgJkRvHcyPe8G6ctLNJthlSWJ48neWS"
python3 feishu_sync.py \
  --config feishu-sync-config.json \
  --data feishu-expo-data.json
```

測試資料格式但不連線飛書：

```bash
python3 feishu_sync.py \
  --config feishu-sync-config.example.json \
  --data feishu-expo-data.json \
  --dry-run
```

同步時會依 `活動名稱` 合併資料，並保留：

- 是否參展
- 負責人
- 預算
- 內部備註
